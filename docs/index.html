Full decision trees of the trained Tsetlin Machines

Each link opens the complete learned logic of one model — every class, every rule (clause), and every condition, with nothing pruned or summarised.

These trees show the model exactly as trained. A Tsetlin Machine is not one decision tree — it is many rules voting at once. Each rule is a soft AND-checklist of conditions; rules that fire cast votes for or against a class; the highest-scoring class wins. Rule firing rates and class sample counts are measured on the real test set. Open any model and use expand / collapse and the filter box to navigate.
