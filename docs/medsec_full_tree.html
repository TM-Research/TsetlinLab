How to read this tree

a class the model can predict
votes FOR the class (add to its score)
votes AGAINST the class (subtract from its score)
a single condition (a check on one reading)
an impossible condition (can never be true)

A rule (clause) is a checklist of conditions joined by AND. It is a soft AND: the rule still counts if a few checks miss. Each rule says how many of its checks must hold to fire, and the fired on N pill shows how many real test samples actually triggered it.

score(class) = Σ (votes FOR that fired) − Σ (votes AGAINST that fired)   →   the class with the highest score wins
