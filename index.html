TsetlinLab

An interactive, in-browser lab for interpretable Tsetlin Machines. View every rule in plain language, trace any sample end-to-end (inputs → bits → clauses fired → class vote → decision), and compare binarisers and engines side by side. Inference runs fully client-side in WebAssembly — no server, no install.

WebAssembly inferencerule-level interpretability4 IoT/security datasetsbinariser × engine comparisonzero backend

What you can do

🔎

See every rule

Each clause shown in plain language — which features must be high or low, the thresholds, and the votes it casts for or against a class.

🧭

Trace any sample

Click a sample for a fully-visual decision path: readings → bits → which clauses fired → per-class vote → final prediction.

🌡️

Inspect the booleaniser

Watch one feature become thermometer bits and literals, live, on the Encoding tab — the GLADE binarisation made explicit.

⚙️

Runs in your browser

The model is recomputed client-side in WebAssembly, bit-exact to the trained model. Open the page and it just works.
